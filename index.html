☁ Supabase 데이터 이전

현재 PC의 localStorage 데이터를 Supabase로 일괄 이전합니다. 중복 데이터는 자동으로 제외됩니다.

☁ 클라우드 백업 (Supabase Storage)

매 영업일(월~금) 23:59(KST)에 서버에서 자동 백업됩니다.
아래 버튼으로 지금 즉시 클라우드에 백업본을 생성할 수 있습니다.

↓ JSON 전체 백업

현재 Supabase에 저장된 모든 데이터를 JSON 파일로 내보냅니다.

↑ JSON 데이터 복원

백업 JSON 파일을 업로드하여 Supabase 데이터를 복원합니다.
